emuLVC

 
 
 
About:
 
emuLVC is an online and offline Web-App for labeling, visualizing and correcting speech and derived speech data. (still in alpha!!)
General usage:
 
Moving the mouse over tiers automatically tracks the mouse movement and calculates the nearest boundary to the current mouse position and preselects it. To move the currently preselected boundary hold down the {{keyShift}} key and move the mouse to the desired position. The boundary will then track the mouse movement. If a segment is selected (= marked yellow), holding down {{keyAlt}} will move the entire segment
Main Navigation Shortcuts:
 
{{openSubmenu}}:Open Submenu
{{keyZoomIn}}:Zoom in1
{{keyZoomOut}}:Zoom out1
{{shiftViewPortLeft}}:Move view right
{{shiftViewPortRight}}:Move view left
{{keyZoomAll}}:View all
{{keyZoomSel}}:Zoom in to selected
{{keyTab}}:Select next event
{{keyShift}} and {{keyTab}}:Select previous event
1.) Zooming while a Boundary is selected will keep the position of that boundary fixed
Labeling shortcuts:
 
{{history}}:Go back in history
{{keyEnter}}:Add event / boundary at selected position
{{keyBackspace}}:Deleted selected Segments
{{keyShift}} + {{keyBackspace}}:Deleted current Segment Boundary
{{snapAbove}}:Snap selected boundary to closest boundary in tier above
{{snapBelow}}:Snap selected boundary to closest boundary in tier below
{{snapZero}}:Snap to nearest zero crossing
{{plus}}:Add time to selected segments (expand to right)
{{minus}}:Remove time to selected segments (shrink from right)
{{keyShift}} and {{plus}}:Add time to selected segments (expand to left)
{{keyShift}} and {{minus}}:Remove time to selected segments (shrink from left)
{{tierUp}}:Move tier selection up
{{tierDown}}:Move tier selection down
{{selectSegmentsInSelection}}:Select all Segments in current Viewport Selection
{{snapBoundary}}:Move selection to currently selected boundary
MOUSE DRAGYou can sort Tiers by dragging them up and down
Playback shortcuts:
 
{{playEntireFile}}:Play entire file
{{playSelected}}:Play selected
{{playAllInView}}:Play/Pause in view (also use as general stop for playback)
Correction Tool shortcuts:
 
{{selectFirstContourCorrectionTool}}:Choose first correction tool
{{selectSecondContourCorrectionTool}}:Choose second correction tool
{{selectThirdContourCorrectionTool}}:Choose third correction tool
{{selectFourthContourCorrectionTool}}:Choose fourth correction tool
{{selectNoContourCorrectionTool}}:Choose NO correction tool (choose when not correcting)
