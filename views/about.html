emuLVC

 
emuLVC is an online and offline Web-App for labeling, visualizing and correcting speech and derived speech data. (still in alpha!!)
General usage:
Moving the mouse over tiers automatically tracks the mouse movement and calculates the nearest boundary to the current mouse position and preselects it. To move the currently preselected boundary hold down the shift key and move the mouse to the desired position. The boundary will then track the mouse movement. If a segment is selected (= marked yellow), holding down alt will move the entire segment
Default Navigation Shortcuts:
(Some might be deactivated / overridden)
{{keyZoomIn}}:Zoom in1
{{keyZoomOut}}:Zoom out1
{{shiftViewPortLeft}}:Move view right
{{shiftViewPortRight}}:Move view left
{{keyZoomAll}}:View all
{{keyZoomSel}}:Zoom in to selected
Tab:Select next event
Shift+Tab:Select previous event
1.) Zooming while a Boundary is selected will keep the position of that boundary fixed
Labeling shortcuts:
(Some might be deactivated / overridden)
Enter:Add event / boundary at selected position
Backspace:Remove selected event
Delete:Remove selected boundary
T:Snap selected boundary to closest boundary in tier above
B:Snap selected boundary to closest boundary in tier below
X:Snap to nearest zero crossing
+:Add time to selected segments (expand to right)
Shift++:Add time to selected segments (expand to left)
-:Remove time to selected segments (shrink from right)
Shift+-:Remove time to selected segments (shrink from left)
↑Move tier selection up
↓Move tier selection down
CMove selection to currently selected boundary
Playback shortcuts:
(Some might be deactivated / overridden)
F:Play entire file
R:Play selected
Space:Play/Pause in view (also use as general stop for playback)
Playback shortcuts:
(Some might be deactivated / overridden)
1:Choose first correction tool
2:Choose second correction tool
3:Choose third correction tool
4:Choose fourth correction tool
^:Choose NO correction tool (choose when not correcting)
